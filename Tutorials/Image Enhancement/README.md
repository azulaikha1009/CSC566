# Image Enhancement

## Overview

This laboratory introduces image enhancement techniques to improve image quality and emphasize important features using MATLAB.

## Topics Covered

- Bit-plane slicing
- Object extraction
- Contrast enhancement
- Histogram equalization
- Adaptive histogram equalization

## MATLAB Files

- BitPlaneSlicing.m
- HighlightRockObject.m
- EnhanceDarkBeanImage.m

## Input Images

- cameraman.tif
- grayrock.jpg
- darkbean.jpg

## MATLAB Functions Used

- bitget()
- imadjust()
- histeq()
- adapthisteq()

## Learning Outcomes

- Analyze image bit planes
- Extract objects from images
- Improve image contrast
- Compare enhancement techniques
