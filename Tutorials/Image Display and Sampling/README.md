# Image Display and Sampling

## Overview

This laboratory focuses on displaying indexed images, interacting with images, resizing images, converting bit depth, and understanding image sampling techniques.

## Topics Covered

- Indexed images
- Image display
- Image resizing
- Bit-depth conversion
- Image sampling

## MATLAB Files

- DisplayIndexedImage.m
- HighlightRedLeaves.m
- ResizeAndConvert16Bit.m
- UndersamplingExample.m

## Input Images

- trees.tif
- tree.jpg
- pout.tif

## MATLAB Functions Used

- imread()
- imshow()
- imresize()
- imwrite()

## Learning Outcomes

- Display indexed images
- Resize images
- Convert image bit depth
- Understand image sampling
