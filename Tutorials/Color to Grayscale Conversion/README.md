# Color to Grayscale Conversion

## Overview

This laboratory introduces the fundamentals of digital image representation in MATLAB by reading RGB images and converting them into grayscale and binary images using built-in image processing functions.

## Topics Covered

- Reading digital images
- RGB image representation
- Binary image conversion
- Grayscale conversion
- Image visualization

## MATLAB File

- ConvertRGBToGrayscale.m

## Input Image

- balloons.png

## MATLAB Functions Used

- imread()
- imshow()
- rgb2gray()
- im2gray()
- im2bw()

## Learning Outcomes

- Understand image representation
- Convert color images to grayscale
- Display images using MATLAB
