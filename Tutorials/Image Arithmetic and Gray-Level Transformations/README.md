# Image Arithmetic and Gray-Level Transformations

## Overview

This laboratory demonstrates image arithmetic operations and gray-level transformations used to manipulate image intensity and improve visual quality.

## Topics Covered

- Region of Interest (ROI)
- Image subtraction
- Image negative
- Log transformation
- Gamma correction
- Histogram stretching
- Histogram equalization

## MATLAB Files

- ExtractCoinUsingROI.m
- ImageSubtraction.m
- ImageNegativeTransformation.m
- LogTransformation.m
- GammaCorrection.m
- HistogramStretching.m
- HistogramEqualization.m
- HistogramAdjustment.m

- Perform image arithmetic
- Extract regions of interest
- Apply gray-level transformations
- Enhance image contrast
