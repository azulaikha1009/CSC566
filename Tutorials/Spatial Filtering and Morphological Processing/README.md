# Spatial Filtering and Morphological Processing

## Overview

This laboratory explores spatial filtering techniques and basic morphological image processing for image enhancement and noise removal.

## Topics Covered

- Average filtering
- Boundary handling
- Median filtering
- Noise removal
- Blob detection
- Morphological processing

## MATLAB Files

- AverageFilterBoundaryComparison.m
- RemoveNoiseUsingMedianFilter.m
- MedianFilteringExample.m
- RemoveSmallBlobs.m

## Input Images

- pattern.tif
- Noisy 1.jpg
- Noisy 2.jpg
- Blob.jpg
- eight.tif

## MATLAB Functions Used

- imfilter()
- fspecial()
- medfilt2()
- bwareaopen()
- imbinarize()

## Learning Outcomes

- Compare filtering methods
- Remove image noise
- Apply median filtering
- Remove unwanted objects using morphology
